**Instructions for AI Agent**

1. **Ensure a Clean Git Work Tree**
   - Read `@docs/GIT_END.md` first.
   - The work tree **must be clean** before committing.
   - If there are untracked or unnecessary files:
     - Add them to `.gitignore` if they should be ignored.
     - Otherwise, stage them with `git add .`.
2. **Maintain the Worklog**
   - Follow the template in `@docs/worklog/WORKLOG_AI_INSTRUCTION.md` to update the worklog file.
3. **Update Metadata Files**
   - Review and update relevant details in `CLAUDE.md`, `AGENTS.md`, and `GEMINI.md`.
   - Ensure agent roles, capabilities, and task scopes are correctly reflected.
4. **General Rules**
   - Follow existing repository conventions and structure.
   - Avoid introducing unused files or changing unrelated code.
   - Document any significant changes in the worklog.
